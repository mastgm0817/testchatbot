🏗️ Safety Sage 위젯 테스트

이 페이지는 Safety Sage 챗봇 위젯의 기능을 테스트하기 위한 페이지입니다.

오른쪽 하단에 챗봇 위젯이 표시되어야 합니다.

📋 테스트 항목

✅ 위젯이 페이지에 정상적으로 로드되는지 확인

✅ 챗봇 버튼을 클릭했을 때 채팅 인터페이스가 열리는지 확인

✅ 메시지를 입력하고 전송할 수 있는지 확인

✅ 음성 인식 기능이 작동하는지 확인

✅ 메시지 복사 기능이 작동하는지 확인

✅ 위젯을 닫을 수 있는지 확인
